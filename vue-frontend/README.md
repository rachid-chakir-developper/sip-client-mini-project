# Vue 3 + TypeScript + Vite

> Pour la documentation du projet (installation, démarrage, utilisation), voir le [README principal](../README.md). Pour le widget d'appel SIP situé dans `src/components/phone/`, voir sa [doc dédiée](src/components/phone/README.md).

Ce modèle de projet permet de démarrer rapidement avec Vue 3 et TypeScript sous Vite. Il utilise les composants monofichiers Vue 3 `<script setup>` — voir la [documentation script setup](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) pour en savoir plus.

Pour la configuration recommandée du projet et le support des IDE, voir le [guide TypeScript de Vue](https://vuejs.org/guide/typescript/overview.html#project-setup).
